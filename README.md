# ObserynX — Quotes & Visuals
A collection of what I live, what I think, and what I never say — a quiet corner of my mind written out.

This project generates square, minimalist images containing personal, melancholic, and introspective quotes.
Each image is created with soft textures, subtle vignettes, and a color palette that reflects different emotional tones.

## Features
- Automatic quote rendering

- Dynamic background colors based on quote type

- Soft vignette and grain effects for a cinematic look

- Centered typography with stroke for readability

- Fully automated batch generation

## Structure
- `data/quotes.json` → All quotes with ID and type

- `images/` → Output folder for generated visuals

- `src/` → Python script that builds the images

## Quote Types
- Type 1: Dark, muted tones

- Type 2: Soft greens

- Type 3: Light blues

## Purpose
This repository is a personal archive — a visual diary of thoughts, feelings, and quiet moments that rarely make it into words.
